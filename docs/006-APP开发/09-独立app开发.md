# 独立app开发（beta）

## 免费打包服务
==仅面向专属设备可用数量1000个以上的开发者提供android Apk打包服务==   

APP打包需要提供的资料：  
包名：\<xxx\>.blinker.app  
桌面图标：<512 x 512  PNG格式>  
启动页图片：<2048 x 2048  PNG格式>  

打包后的apk文件未签名，用户自行签名后，可发布到应用商店。
